# TelegramBot
Telegram bot for easy interaction with it. It is completely written in python 3.0. 
Installation is extremely simple: 
just run this script on any device with a python interpreter, having previously installed the necessary libraries.
Work with Chrome browser.
## Libraries used
- **telebot** for the work of the requests bot telegrams
- **selenium** and **BeautifulSoup** for automatic parsing (the bot can find pictures by casting and rhymes to words)
- **matplotlib** for plotting graphs
- **mysql.connector** to connect to mysql database. This is necessary to store some user data
## What the bot can do
- find pictures by request
- create graphs based on the entered mathematical formula
- respond with prepared phrases
- "flip a coin"
- look for rhymes to the entered words
## how you can improve the bot
- replace selenium with html parser to speed up request processing
- add new prepared phrases
- place on permanent hosting
